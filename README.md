# Landing Page SaaS - Apollo Style

Landing page moderna estilo SaaS B2B inspirada en Apollo.io

## Tecnologías

- React 18
- Vite
- JavaScript
- SWC
- Tailwind CSS
- Framer Motion

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
